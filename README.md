## Install

### Via skills.sh (recommended)

```bash
npx skills add yinpengfei/skills
```

### Via ClawHub

```bash
clawhub install dbq
```

### Manual

Download from [GitHub Releases](https://github.com/yinpengfei/skills/releases) and extract to your agent's skills directory.
